### 1. Create virtual environment and install packages
Windows
```shell
python -m venv venv && venv\Scripts\activate && pip install -r requirements.txt
```

Linux/Mac
```shell
python3 -m venv venv && source venv/bin/activate && pip3 install -r requirements.txt
```
